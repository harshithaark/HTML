# CSS

## What is CSS?
CSS stands for Cascading Style Sheets. It describes how HTML elements are to be displayed on screen, paper, or in other media.

## What is the use or purpose of CSS?
It is a style sheet language which is used to describe the look and 
formatting of a document written in markup language. 
It provides an additional feature of HTML. 
It is generally used with HTML to change the Style of web pages and user 
interfaces. 
It is easier to make the web pages presentable using CSS. 
It is easy to learn and understand and used to control the presentation of 
an HTML document. 
CSS helps us to control the text color, font style, the spacing between 
paragraphs, sizing of columns, layout designs, and many more.

## What is the history of CSS? 
1994⇒ ‘ Hakon Wium Lie ’ proposed the idea of CSS. 
Till time there are 3 versions of CSS introduced
Now we are using the latest version of CSS i.e. 
CSS3 
CSS1 (1996): 
The first official specification released in → 
1996 
It allowed basic features. 
Basic Features like ⇒ Styling such as changing text colors, fonts, and 
backgrounds colors. 
Also it provide the limited layout control to HTML document. 
CSS2 (1998): 
The second official specification released in → 
1998 
Introduced with new features. 
New Features such as ⇒ positioning, Z-index, and the introduction of 
media types for different devices. 
Improved control over layout to HTML document. 
CSS3 (2011 - Ongoing): 
The first draft of CSS3 released in → 
1999 
CSS3 is not a single monolithic release but a collection of modular 
specifications introduced gradually. 
Modular Approach ⇒ Divided into modules. 
These modules cover various aspects of web design, such as 
animations, gradients, and flexible box, grid layout, etc. 
CSS4 (2020 - in development): 
CSS4 is the future of CSS, aiming to further enhance web design 
capabilities, including responsive design, variable fonts, and improved 
grid systems.

## How to add CSS to HTML?
- Inline: Using the `style` attribute
Example - `<h1 style='background-color:red;color:green;'></h1>`
- Internal: Using `<style>` tag in `<head>`
- External: Linking a `.css` file with `<link>`

#### Note: The first priority is inline CSS, then external CSS, and lastly internal CSS
