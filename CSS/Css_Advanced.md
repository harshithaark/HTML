# CSS Advanced

## What are CSS Animations?
CSS animations change property values over time using `@keyframes`.

## What are CSS Transitions?
Transitions smoothly change property values when triggered.

## What are CSS Transforms?
`transform` property changes element shape, size, position:
- `translate()`: Move
- `rotate()`: Rotate
- `scale()`: Resize
- `skew()`: Distort

## What are CSS Media Queries?
Media queries apply styles based on device characteristics like screen size.

## What is CSS Responsive Design?
Responsive design makes websites work on all devices using flexible layouts and media queries.

## What is CSS Variables?
CSS custom properties (variables) store reusable values.

## What is CSS Specificity?
Specificity determines which CSS rule applies when multiple rules target the same element.

## What is CSS Inheritance?
Inheritance allows child elements to inherit parent styles.

## What is CSS Cascade?
Cascade determines which styles apply when multiple sources exist.

## What is CSS Pseudo-Classes?
Pseudo-classes style elements in specific states like `:hover`, `:focus`.

## What is CSS Pseudo-Elements?
Pseudo-elements style parts of elements like `::before`, `::after`.

## What is CSS Vendor Prefixes?
Vendor prefixes like `-webkit-` ensure compatibility with different browsers.

## What is CSS Preprocessing?
CSS preprocessors like Sass extend CSS with variables, nesting, mixins.

## What is CSS Postprocessing?
Postprocessors like Autoprefixer add vendor prefixes automatically.

## What is CSS Frameworks?
Frameworks like Bootstrap provide pre-built components and utilities.
