## Key HTML Concepts

### Structure
Every HTML document has a basic structure:
- `<!DOCTYPE html>` - Declares the HTML version
- `<html>` - Root element
- `<head>` - Contains metadata and title
- `<body>` - Contains visible content

### Semantic HTML
Use meaningful tags like `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<footer>` instead of generic `<div>` tags. This improves accessibility and SEO.

### Attributes
Elements can have attributes that modify their behavior:
- `id` - Unique identifier for an element
- `class` - For styling groups of elements
- `href` - Link destination
- `src` - Image or media source
- `alt` - Alternative text for images

### Content Types
- **Text content** - Formatted with tags like `<p>`, `<h1>-<h6>`, `<strong>`, `<em>`
- **Links** - Created with `<a>` tags
- **Images** - Embedded with `<img>` tags
- **Lists** - Ordered `<ol>`, unordered `<ul>`, or definition `<dl>`
- **Tables** - Structured data with `<table>`, `<tr>`, `<td>`
- **Forms** - User input with `<form>`, `<input>`, `<button>`
- **Multimedia** - Video `<video>` and audio `<audio>` elements
