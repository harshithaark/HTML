# HTML-CSS-JS-and-React

This repository contains comprehensive documentation and guides for learning HTML, CSS, JavaScript, and React. It's structured to help beginners progress from basics to advanced topics.

## Getting Started

### HTML (HyperText Markup Language)
HTML provides the structure and content of web pages. Start here if you're new to web development.

**Where to start:**
1. **[HTML Basics](HTML/Html_Basics.md)** - Learn the fundamental structure, tags, DOCTYPE, and basic HTML elements
2. **[Web Theory](HTML/Web_theory.md)** - Understand how the web works, including HTTP, DNS, and browser components
3. **[HTML Attributes](HTML/Html_Attributes.md)** - Learn about HTML attributes like id, class, src, alt, etc.

For a complete learning path, see the [HTML Complete Guide](HTML/Html.md).

### CSS (Cascading Style Sheets)
CSS controls the presentation and styling of HTML elements. Learn CSS after you have basic HTML knowledge.

**Where to start:**
1. **[CSS Basics](CSS/Css_Basics.md)** - Understand CSS syntax, selectors, and basic styling properties
2. **[CSS Layout](CSS/Css_Layout.md)** - Learn about positioning, display properties, and layout techniques
3. **[CSS Flexbox](CSS/css_flexbox.md)** - Master flexible box layout for modern responsive designs

For an overview of all CSS topics, see the [CSS Documentation](CSS/css.md).

## Repository Structure
- `HTML/` - HTML documentation and guides
- `CSS/` - CSS documentation and guides
- `Images/` - Supporting images for the documentation

## Next Steps
After mastering HTML and CSS basics:
- Learn JavaScript for interactivity
- Explore React for building dynamic user interfaces
- Practice by building small projects
